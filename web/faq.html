1.1. which difference with repackage option in recent rpm releases
when using rpm repackage option, you have to uninstall the package, not with rpmrebuild.

1.2. error: No compatible architectures found for build
It occurs, for example, when you have a linux distribution on 64 bits, and try to rebuild a 32 bit package. A nice way to solve the problem is to use the setarch command : setarch i586 rpmrebuild your_package
